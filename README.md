### [mer](https://www.youtube.com/watch?v=EoQW03UFqQw)

Grab a random commit message from GitHub.


#### Install

`sudo make install`

#### Get started

Set your own API key in the env variable:

```
token = os.environ.get('YOUR_GITHUB_API_KEY')
```

`mer -c or --commit`
